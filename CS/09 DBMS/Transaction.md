### Transaction
- Transaction is a set of operations which are all logically related
#### Main operations:
- Read 
- Write
- READ - read from DB into buffer in main memory
- WRITE - write from buffer to DB

---
### States of transaction
- Active state
- Partially committed state
- Committed state
- Failed state
- Aborted state
- Terminated state
